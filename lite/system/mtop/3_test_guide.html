4.4.3.2. 编译测试工具¶

MTOP测试工具可以按以下配置打开：

Drivers options --->
    Drivers examples --->
        [*] Enable mtop driver test command

配置后，重新编译。

4.4.3.3. mtop测试¶

可以直接调用 mtop 命令进行测试。（默认配置1s更新一次参数）。

该命令的其它参数可以参考help命令：

aic /> mtop -h

    Usage: mtop [-n iter] [-d delay] [-h]
    -n NUM   Number of updates before this program exiting.
    -d NUM   Seconds to wait between update.
    -h Display this help.
